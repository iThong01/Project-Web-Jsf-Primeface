# Project Context: GreenMarketJsf

## Technology Stack
- **Frameworks/Libraries:** This web application is built strictly using **JSF** and **PrimeFaces**.
- **Allowed Tools:** You may only use **PrimeFaces**, **PrimeFlex**, **PrimeBlocks**, and their built-in icons.
- **Strict Constraint:** Do NOT use any other front-end libraries or frameworks.
- **Strict Constraint:** Do NOT write custom CSS or styling unless it is absolutely necessary (i.e., when PrimeFaces/PrimeFlex cannot solve the problem).

## Interaction and Learning Guidelines (Crucial)
- **Step-by-Step Guidance:** Whenever adding features or modifying code, guide the user step-by-step. Do not provide large blocks of code all at once.
- **Detailed Explanations:** Explain what each line or block of code does and *why* it is being used.
- **No Direct File Modifications:** Do NOT modify or write to files directly automatically. Always present the instructions and code to the user so they can apply the changes themselves. The user is actively learning the process.
